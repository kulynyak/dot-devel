# dot-devel

development configuration configuration

## Install
This dot files config is available as [Ellipsis][Ellipsis] package.

#### Ellipsis
``` shell
    ellipsis install kulynyak/dot-devel
```

## Maintain
To keep everything up-to-date;

#### Ellipsis
```shell
    ellipsis pull devel
```
